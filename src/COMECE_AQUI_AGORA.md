## 🎯 OBJETIVO

---

## 📋 CHECKLIST RÁPIDO

### ✅ **PASSO 1: GITHUB**
```
https://github.com/login
→ Fazer login (ou criar conta)
```

### ✅ **PASSO 2: REPOSITÓRIO**
```
GitHub → Repositories → volleypro
OU
https://github.com/[SEU-USUARIO]/volleypro
```

### ✅ **PASSO 3: CRIAR CODESPACE**
```
Botão verde "Code"
→ Aba "Codespaces"
→ "Create codespace on main"
→ AGUARDAR 2-3 MINUTOS
```

### ✅ **PASSO 4: VS CODE CARREGOU!**
```
Você verá VS Code completo online
Explorer à esquerda
Editor no centro
Terminal embaixo
```

### ✅ **PASSO 5: TESTE**
```
Explorer → App.tsx
Mudar algum texto
Ctrl+S (salvar)
```

### ✅ **PASSO 6: PUBLICAR**
```
Terminal (Ctrl+`)
→ bash publicar.sh
→ ENTER
→ Aguardar 5min
→ Acessar: https://volleypro-zw96.vercel.app
→ Ctrl+Shift+R (limpar cache)
→ ✅ ATUALIZADO!
```
